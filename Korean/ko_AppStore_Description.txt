설명:

Amphetamine에 오신걸 환영합니다. 이제껏 macOS에 있어서 가장 강력한 잠깨우기 앱이죠. Amphetamine은 Mac을 재우지 않습니다. 덤으로 디스플레이도 말이죠. 그러면서도 매우 간단한 활성화/비활성화 스위치와 더불어 자동으로 처리해주는 쉽게 편집가능한 트리거도 있지요. Amphetamine은 매우 강력하면서 여러 전문적인 기능을 가지고 있지만, 여전히 모든 이들이 편하게 쓸수 있는 도구입니다.

=== 클릭하여 Mac을 깨우세요 ===

Amphetamine의 메뉴에서 빠르게 Mac을 깨울수 있습니다.

• 무기한, 혹은 지정한 시간 아님 특정한 시간까지
• 파일을 다운로드 하는 경우
• 특정 앱을 실행하는 경우

깨어있는 세션 동안 다음과 같은 행동들을 하거나 막을수 있습니다:

• 디스플레이 잠자기
• 화면 보호기 동작
• 내장 모니터가 닫혔을시 시스템 잠자기
• 자동으로 마우스 커서 움직이기
• 스크린 잠그기

=== 진보한 자동화 ===

트리거를 설정하여 자동으로 Mac을 깨우세요. 

• 외부 디스플레이가 연결되었을시
• Mac 내장 디스플레이가 미러링 되었을때
• USB 혹은 Bluetooth 장비가 연결되었을때
• 특정한 앱이 실행중일때
• 특정한 앱이 실행중이면서 앞으로 나온 경우
• Mac의 배터리가 충전중이거나 혹은 배터리 충전량이 이상일 경우
• Mac의 전원 어뎁터가 연결되었거나 연결되지 않았을때
• Mac이 특정 IP에 연결되었을때
• Mac이 특정 Wi-Fi 네트워크에 연결 되었을때
• Mac이 Cisco AnyConnect VPN 서비스에 연결되었을때
• Mac이 특정 DNS 서버에 연결되었을때
• 헤드폰이나 기타 오디오 출력이 연결되었을때
• 특정한 드라이브 혹은 볼륨이 마운트 되었을때
• Mac의 CPU 사용량이 특정 값을 넘을때
• 특정시간 동안 Mac이 유휴 시간에 있을경우


=== 그래서 호구 잡히는거 있죠? ===

그런거 없습니다. Amphetamine은 100% 공짜이며, 여러분에게 드리는 선물입니다. 잘 즐기세요!

• 광고 없음
• 인앱 결재 없음
• 잠금/프로 버전 없음
• 리뷰 좀 써달라고 조르는거 없음
• 다른 유료앱으로 가는 링크 없음
• 트래킹 없음
• 데이터 하베스팅 없음


=== 그래서 Amphetamine으로 뭘 할수 있죠? === 

Amphetamine은 macOS에서 이제까지 나온 것중에서 강력하면서도 수많은 사용자 정의 기능들로 가득찬 잠깨우기 유틸리티입니다. 몇몇 유명한 기능들이 여기 있습니다:

• 어느정도 시간동안 만지지 않으면 자동으로 Mac 스크린 잠금
• 시간마다 마우스 커서 움직이기
• 디스플레이/뚜껑이 덮혀 있어도 깨어있기
• Drive Alive를 이용한 항상 깨어있는 드라이브
• AppleScript 지원
• 여러분만의 메뉴 바 아이콘 지정 가능
• 여러분 만의 알림 소리 가능
• Mac이 깨어 있어도 디스플레이는 잠 들수 있게 하기
• 어느정도 시간동안 만지지 않으면 지정한 시간에 화면 보호기 띄우기
• 메뉴에서 세션 타임 보이기 설정가능(12/24시간 제)
• Mac 배터리가 부족하다면 세션을 자동 종료
• 세션 시작과 종료, 디스플레이 잠자기를 활성화/비활성화 등등 할수 있는 핫키 지원
